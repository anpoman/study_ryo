Firebase の設定
初回のみ設定が必要です。以下の手順で Firebase プロジェクトを作成してください。
1

Firebase プロジェクトを作成

console.firebase.google.com にアクセスし、Google アカウントでログイン後「プロジェクトを追加」をクリック。プロジェクト名を入力して作成します。

2

ウェブアプリを登録

プロジェクト画面で </>（ウェブ）アイコンをクリック。アプリのニックネームを入力して「アプリを登録」を押す。

3

設定コードをコピー

表示される firebaseConfig = { ... } の { } の中身全体をコピーして下のテキストエリアに貼り付けてください。

4

Firestore と認証を有効化

左メニューの「Firestore Database」→「データベースを作成」→「テストモード」で開始。
「Authentication」→「Sign-in method」→「メール/パスワード」と「Google」を有効にする。

空欄のまま保存するとIDが自動生成されます。設定後にIDをコピーして他のデバイスで入力することでデータを共有できます。
